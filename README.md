# HR-Insight-Dashboard
An AI-powered employee dashboard that predicts attrition risk and promotion likelihood using machine learning. Update profiles, track tasks with live pie charts, generate PDF reports with embedded visualizations, and export data to Excel and a job applicant portal. Built with Flask, scikit-learn, matplotlib, and fpdf.
# HR Insight Bot

AI-powered HR dashboard with **attrition & promotion predictions**, **PDF reports**, **task tracker**, and **applicant portal**.

---

## Features
- Predicts **attrition risk** & **promotion chance** (ML)
- Generates **PDF reports with charts**
- Task tracker with **live pie chart**
- Excel export (`hr_data.xlsx`)
- Secure login with password hashing
- Applicant job application system
